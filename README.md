# 📊 TABLES USED IN THE PROJECT

## 1️⃣ COUNTRY TABLE

Stores country information.

```sql
CREATE TABLE country (
    CID VARCHAR(10) PRIMARY KEY,
    Country VARCHAR(100) UNIQUE
);
```

### PURPOSE
- Central table for all country-related references
- Maintains unique country records

---

## 2️⃣ EMISSION TABLE

Stores carbon emission data for different energy types.

```sql
CREATE TABLE emission_3 (
    country VARCHAR(100),
    energy_type VARCHAR(50),
    year INT,
    emission INT,
    per_capita_emission DOUBLE,
    FOREIGN KEY (country) REFERENCES country(Country)
);
```

### PURPOSE
- Stores total emissions
- Tracks per capita emissions
- Analyzes pollution trends

---

## 3️⃣ POPULATION TABLE

Stores population information for countries.

```sql
CREATE TABLE population (
    countries VARCHAR(100),
    year INT,
    Value DOUBLE,
    FOREIGN KEY (countries) REFERENCES country(Country)
);
```

### PURPOSE
- Tracks population growth over time
- Used for comparative analysis with emissions and GDP

---

## 4️⃣ PRODUCTION TABLE

Stores energy production data.

```sql
CREATE TABLE production (
    country VARCHAR(100),
    energy VARCHAR(50),
    year INT,
    production INT,
    FOREIGN KEY (country) REFERENCES country(Country)
);
```

### PURPOSE
- Tracks energy production by country and energy type
- Used to compare production trends globally

## 5️⃣ GDP TABLE

Stores GDP data for countries.

```sql
CREATE TABLE gdp_3 (
    Country VARCHAR(100),
    year INT,

    FOREIGN KEY (Country) REFERENCES country(Country)
```

### PURPOSE
- Stores economic growth information
- Used for GDP and energy consumption analysis

## 6️⃣ CONSUMPTION TABLE

```sql
CREATE TABLE consumption (
```

## 🔹 GENERAL & COMPARATIVE ANALYSIS

### Total Emission Per Country

```sql
SELECT country, SUM(emission) AS total_emission
FROM emission_3
WHERE year = (SELECT MAX(year) FROM emission_3)
GROUP BY country
ORDER BY total_emission DESC;
```

### Top 5 Countries by GDP

```sql
SELECT country, value AS gdp
FROM gdp_3
WHERE year = (SELECT MAX(year) FROM gdp_3)
ORDER BY gdp DESC
LIMIT 5;
```

### Energy Production vs Consumption

```sql
SELECT p.country, p.year, p.energy, p.production, c.consumption
FROM production p
JOIN consumption c 
ON p.country = c.country 
AND p.year = c.year 
AND p.energy = c.energy;
```

# 📉 TREND ANALYSIS

The project includes multiple trend analysis queries to study how energy and economic indicators change over time.

### Global Emission Trend

```sql
SELECT year,
       SUM(emission) AS total_global_emission
FROM emission_3
GROUP BY year
ORDER BY year;
```

### Global GDP Trend

```sql
SELECT year,
       ROUND(SUM(Value), 2) AS global_gdp
FROM gdp_3
GROUP BY year
ORDER BY year;
```

### Global Population Trend

```sql
SELECT year,
       ROUND(SUM(Value), 0) AS global_population
FROM population
GROUP BY year
ORDER BY year;
```

# ⚖️ RATIO & PER CAPITA ANALYSIS

The project also performs ratio analysis to compare energy consumption relative to GDP and population.

### Consumption Relative to GDP

```sql
SELECT c.country,
       c.year,
       SUM(c.consumption) AS consumption,
       g.Value AS gdp,
       ROUND(SUM(c.consumption)/g.Value,6) AS ratio
FROM consumption c
JOIN gdp_3 g 
ON c.country = g.Country 
AND c.year = g.year
GROUP BY c.country, c.year, g.Value
ORDER BY ratio DESC;
```

### Highest Per Capita Emission

```sql
SELECT country, per_capita_emission
FROM emission_3
WHERE year = (SELECT MAX(year) FROM emission_3)
ORDER BY per_capita_emission DESC
LIMIT 1;
```
